This folder contains software associated with the Amstrad CPC CTC DART.

* siolif01.asm contains function libary associated with the DART.
* siotst10.asm contains a test program in assembler
* siotst03.bas contains a simple test program in BASIC
* siotst07.bas contains a send an receive test program in BASIC
* siotst01.dsk disk file containing siotst03.bas and siotst07.bas
